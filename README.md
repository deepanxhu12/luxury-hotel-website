# 🏨 Luxury Hotel Website

A modern and elegant luxury hotel landing page built using **HTML**, **CSS**, and **JavaScript**. The website focuses on delivering a premium user experience with smooth animations, interactive sections, and a clean luxury-inspired design.

## ✨ Features

- Premium Hero Section
- Luxury Hotel Showcase
- Interactive Video Section
- Room Showcase with Pricing
- Custom Hover Effects
- Cursor Follow Animation
- Luxury Navigation Menu
- Booking Section
- Modern UI Design
- Smooth CSS Animations

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/deepanxhu12/luxury-hotel-website.git
```

2. Open the project folder

```bash
cd luxury-hotel-website
```

3. Open `index.html` in your browser.
